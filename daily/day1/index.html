CSS 元素與顏色示範

我是P段落，文字顏色是什麼呢? 區塊元素還是行內元素呢? ==> 綠色，區塊元素

我是span，文字顏色是什麼呢? 區塊元素還是行內元素呢? ==> 綠色，行內元素
我是div，文字顏色是什麼呢? 區塊元素還是行內元素呢? ==> 紅色，區塊元素
我是a連結，文字顏色是什麼呢? 區塊元素還是行內元素呢? ==> 綠色，行內元素
我是img，區塊元素還是行內元素呢? ==> 行內元素

我們知道在 CSS 類別套用的優先順序 —— 在同等優先權下，後宣告的類別會覆蓋前面的，但類別在 HTML 中的順序不會影響這個行為。

  • p - 為綠色，區塊為 block
  • span - 為綠色，區塊為 inline
  • div - 為紅色，區塊為 block
  • a - 為綠色，區塊為 inline
  • img - 區塊為 inline
