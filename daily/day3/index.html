box1預設為content-box

內容寬度：300px

padding 左右：16px + 16px = 32px

border 左右：3px + 3px = 6px

總寬度 = 300 + 32 + 6 = 338px

Content-box：padding 和 border 是額外加上去的

box2設定為border-box

總寬度：300px

padding 左右：16px + 16px = 32px

border 左右：3px + 3px = 6px

內容寬度 = 300 - 32 - 6 = 262px

Border-box：寬度包含 padding 和 border

Content-box 高度計算

line-height：24px

padding 上下：16px + 16px = 32px

border 上下：3px + 3px = 6px

總高度 = 72 + 32 + 6 = 110px

因未設定 height，會根據內容自動撐開

Border-box 高度計算

line-height：24px

padding 上下：16px + 16px = 32px

border 上下：3px + 3px = 6px

總高度 = 72 + 32 + 6 = 110px

因未設定 height，會根據內容自動撐開

視覺比較
Content-box
寬度 = 內容寬度
Border-box
寬度 = 內容寬度 + Padding + Border
